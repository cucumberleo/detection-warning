| ---------- | -------------------------------------------------------------- | ---------------------------------------------------- | ------ | --------------------------------------------------------------- |
| 登录与会话 | 用户登录、退出、会话恢复、登录过期处理                         | 管理员登录、退出、管理员路由守卫                     | P0     | 刷新后状态正确；未登录返回登录页；用户不能进入管理页面          |
| 权限控制   | 只能访问本人被授权的设备、检测记录和预警                       | 可以访问全局数据并执行管理操作                       | P0     | 权限必须由后端校验，不能只隐藏前端按钮                          |

| 设备列表   | 查看授权设备的名称、材料、位置、在线状态和信号质量             | 新增、编辑、停用设备，管理材料、位置、状态和设备授权 | P0     | 用户看不到未授权设备；离线设备不可发起检测                      |
| 检测控制   | 选择授权设备和目标气体，发起检测，显示处理状态                 | 查看所有设备检测状态，按设备发起诊断或复测           | P0     | 防止重复提交；失败可重试；记录算法版本与处理耗时                |
| 响应曲线   | 展示检测曲线、时间轴、单位、峰值和最终检测结果                 | 查看任意设备曲线，比较不同设备或检测批次             | P0     | 当前支持静态结果；接入网关后升级为 WebSocket/SSE 实时曲线       |
| 检测结果   | 显示气体类型、浓度、置信度、状态、算法版本                     | 查看全局结果及来源设备、用户、模型版本和异常原因     | P0     | 页面展示必须与检测结果契约一致，不能在前端重新判定风险          |
| 用户预警   | 查看自己的预警，进入详情，确认预警并填写处理备注               | 查看全局预警，分配责任人、确认、关闭、添加备注       | P0     | 完整记录发生、确认、关闭时间和操作者                            |
| 阈值管理   | 只能查看当前生效阈值，不允许修改                               | 编辑不同气体阈值和风险等级，查看修改历史             | P0     | 阈值修改需要校验和二次确认；历史记录不重新计算                  |
| 历史记录   | 按日期、设备、气体、状态筛选，查看检测详情                     | 全局筛选、分页、设备/用户/模型版本查询               | P0     | 筛选条件可组合；分页稳定；时间与单位统一                        |
| 趋势分析   | 浓度趋势、峰值、平均值、预警次数和气体分布                     | 全局趋势、设备对比、材料对比、模型版本对比           | P1     | 图表有空状态、图例、单位和数据时间范围                          |
| 数据导出   | 导出本人可见范围内的 CSV                                       | 按筛选条件导出全局 CSV 或统计报告                    | P1     | 导出权限由后端控制；导出内容与当前筛选一致                      |
| 用户管理   | 无                                                             | 新增、编辑、停用用户，配置角色和设备授权             | P0     | 授权变化立即生效；操作写入审计日志                              |
| 设备授权   | 查看自己当前被授权的设备                                       | 将设备分配给用户，撤销授权，查看授权关系             | P0     | 一个用户可以授权多台设备；撤销后不能继续读取数据                |
| 通知中心   | 查看未读预警、标记已读、设置通知偏好                           | 配置通知策略、查看通知发送状态和失败原因             | P1     | 支持站内通知；PWA/Web Push 可作为后续增强                       |
| Agent 界面 | 提问本人设备、趋势和预警；显示工具状态及数据范围               | 查询全局设备、预警和统计；显示工具调用和数据范围     | P1     | 前端只负责聊天界面；Agent、工具和算法由你负责；Agent 不直连 SQL |
| 通用建议   | 展示基于当前数据的处置建议                                     | 查看平台级运行建议                                   | P2     | 不显示没有真实依据的“个性化分数”，功能需标明数据来源            |
| 审计日志   | 查看自己的重要操作记录                                         | 查看登录、授权、阈值、预警和设备管理日志             | P1     | 日志不可由普通用户修改；支持时间和操作类型筛选                  |
| 系统状态   | 显示服务是否正常、数据更新时间                                 | 查看 API、数据库、设备网关和算法服务状态             | P1     | 不能只显示固定“正常”；必须来自真实健康检查接口                  |
| 用户偏好   | 语言、默认设备、默认气体、通知设置                             | 平台默认配置和功能开关                               | P2     | 设置持久化，重新登录后仍然有效                                  |
| 响应式布局 | 完整支持手机、平板和桌面                                       | 管理表格重点支持桌面，同时保证移动端可操作           | P1     | 390px 宽度无横向溢出；移动端所有入口均可访问                    |
| 可访问性   | 键盘导航、表单标签、ARIA、颜色之外的状态文字                   | 管理表格、弹窗、表单和危险操作可键盘操作             | P1     | 核心流程通过键盘完成；错误信息可被辅助技术识别                  |
| 自动化测试 | 用户登录、检测、历史、预警和越权测试                           | 管理员登录、用户管理、设备管理、阈值和权限测试       | P0     | 单元测试、接口测试、关键流程 E2E 全部通过                       |
